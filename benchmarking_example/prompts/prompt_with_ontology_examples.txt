You are a knowledge engineer and a climate change scientist. I need you to model a climate change variable using the I-ADOPT framework. A variable is usually the result of a sensor measurement or of a laboratory analysis on a sample. The task is to generate Turtle code with a description of the variable according to the I-ADOPT ontology.

example 1:

input variable: aphids incidence in an agricultural experimental plot

output in turtle:

```
@prefix envthes: <http://vocabs.lter-europe.net/EnvThes/> .
@prefix skos: <http://www.w3.org/2004/02/skos/core#> .
@prefix iadopt: <https://w3id.org/iadopt/ont/> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .
@prefix owl: <http://www.w3.org/2002/07/owl#> .
@prefix dc: <http://purl.org/dc/terms/> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .
@prefix : <https://w3id.org/iadopt/ont/> .

iadopt:Variable
  rdfs:label "Variable" ;
  a owl:Class .
  
envthes:30326
  iadopt:hasProperty :incidence ;
  iadopt:hasMatrix :agricultural_experimental_plot ;
  skos:prefLabel "aphids incidence in an agricultural experimental plot"@en ;
  a iadopt:Variable, skos:Concept ;
  skos:inScheme envthes: ;
  iadopt:hasObjectOfInterest :aphids .  

iadopt:hasProperty
  rdfs:comment "A Variable has a Property that characterizes an Entity." ;
  rdfs:label "hasProperty" .

iadopt:hasMatrix
  rdfs:subPropertyOf iadopt:hasContextObject ;
  rdfs:comment "A Variable might have an Entity in which the ObjectOfInterest is contained." ;
  rdfs:label "hasMatrix" .
  
iadopt:hasObjectOfInterest
  rdfs:comment "A Variable has an Entity whose Property is observed." ;
  rdfs:label "hasObjectOfInterest" .

:incidence a iadopt:Property ;
	rdfs:label "Incidence" ;
	rdfs:comment "The relative frequency of occurrence of something." .
	
:agricultural_experimental_plot a iadopt:Entity  ;
	rdfs:label "agricultural experimental plot" ;
	rdfs:comment "A site within which an agricultural experimental process is conducted" .
	
envthes:
  skos:prefLabel "EnvThes"@en ;
  a owl:Ontology, skos:ConceptScheme .
 
:aphids a iadopt:Entity  ;
	rdfs:label "Aphids" ;
	rdfs:comment "" .
```
